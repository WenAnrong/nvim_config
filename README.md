
# nvim_configuration

这是我的nvim和coc.nvim的配置文件，也就是个人的一个备份文件

### 配置

如果使用本配置，需要安装的一些东西：

#### for coc.nvim
```shell
sudo npm install -g neovim
sudo pip3 install pynvim
sudo pip3 install pylint
sudo npm install -g typescript
sudo pip3 install jedi
```

#### for tagbar
```shell
sudo dnf install ctags
```
